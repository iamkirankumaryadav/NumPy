# `Method` | `Function`

### `np.array()`

```python
print(np.array([1,2,3,4]))

[1 2 3 4]
```

### `np.arange()`

```python
print(np.arange(2, 10, 2))

array([2,4,6,8])
```

### `np.unique()`

```python
print(np.unique(np.array([1,1,1,2,2,3])))

array([1,2,3])
```

### `np.random.shuffle()`

```python
a = np.array([0,1,2,3,4,5])

np.random.shuffle(a)

array([4,0,5,1,3,2])
```
